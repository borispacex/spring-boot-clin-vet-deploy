# Project SpringBoot for deploy

## Construir la imagen
```
docker.compose build
docker-compose -f docker-compose.yml build
```
## Levantar contenedores
```
docker-compose up
docker-compose up -d
```
## Ir la URL (Comprobar)
- http://localhost:8080/

## Configurar server DonWeb o cualquier otro VPS
- Instalar docker
